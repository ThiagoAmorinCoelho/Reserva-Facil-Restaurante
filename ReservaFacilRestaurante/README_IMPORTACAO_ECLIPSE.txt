COMO ABRIR NO ECLIPSE IDE:
1. Abra o Eclipse.
2. Clique em File > Import.
3. Escolha General > Existing Projects into Workspace.
4. Marque Select archive file.
5. Selecione o arquivo ReservaFacilRestaurante_Eclipse.zip.
6. Clique em Finish.
7. Abra src > restaurante > Teste.java.
8. Clique com o botao direito em Teste.java > Run As > Java Application.

CLASSE PRINCIPAL:
restaurante.Teste

OBJETOS JA CONSTRUIDOS PARA DEMONSTRACAO:

SUGESTAO DE APRESENTACAO:
1. Rodar o projeto e mostrar os dados da entrega na opcao 1.

7. Tentar reservar uma mesa com mais pessoas que a capacidade para mostrar o erro de capacidade.
